# parkometer_backend
backend for capstone project

After downloding the proogect be sure to run the comand "npm i" in the project's root directory in order to install all necessary node modules, make sure that mongodb is is properly set up on your device, and that check that ports 3000 and 27017 arent in use.

In order to run the project run the comand "npm start" in the project's root directory, you can then make http resquest at http://localhost:3000.

To test the program just run "npm test".

All request can be found in src/router.ts and the code they exacute is in src/controlers.
